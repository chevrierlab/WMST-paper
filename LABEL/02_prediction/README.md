# Validation and Classification

This folder contains notebooks for feature extraction and classification using the UNI model.

- `01_Feature_extraction_Validation.ipynb` – Extracts UNI features for validation.
- `02_KnnClassifier_Validation.ipynb` – Runs kNN classification on validation data.
- `03_KnnClassifier_Additional.ipynb` – Applies kNN to additional datasets.
- `04_KnnClassifier_LOOCV.ipynb` – Performs leave-one-out cross-validation with kNN.
- `uni/` – UNI model resources.
- `assets/` – UNI model related resources.
