# Visualization Notebooks

This folder contains notebooks for visualizing classification and validation results produced with the UNI model.

- **`01_TestSetVisulizationKNN.ipynb`**
  - Visualizes kNN classification results on the test set.
  - Provides plots to compare predictions with ground truth.
- **`02_AdditionalSlicesVisulization.ipynb`**
  - Displays results on additional tissue slices beyond the main dataset.
  - Useful for checking generalization and consistency across samples.
- **`03_LOOCV_TestSetVisulizationKNN_full.ipynb`**
  - Visualizes leave-one-out cross-validation (LOOCV) results.
  - Offers a full overview of classification outcomes across all held-out sets.
